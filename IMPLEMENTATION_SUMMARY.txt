6. معايير التوافق
═══════════════════════════════════════════════════════════════════════════════════

✅ RTL (اليمين إلى اليسار)
   - دعم كامل للغة العربية
   - جميع العناصر تتجاوب مع اتجاه النص

✅ Responsive Design
   - Desktop: 1280px+
   - Tablet: 768px - 1279px
   - Mobile: < 768px

✅ التوافق مع Salla
   - جميع مكونات Salla محفوظة
   - لا تغيير في البنية الديناميكية
   - جميع الإعدادات الأصلية محفوظة

═══════════════════════════════════════════════════════════════════════════════════
7. خطوات التثبيت
═══════════════════════════════════════════════════════════════════════════════════

1. التثبيت:
   cd theme-altour-aldahabi
   pnpm install

2. البناء:
   pnpm run production  (للإنتاج)
   pnpm run development (للتطوير)
   pnpm run watch       (المراقبة المستمرة)

3. التثبيت في Salla:
   - رفع الثيم إلى لوحة Salla
   - تفعيل الثيم من الإعدادات
   - تخصيص الألوان والخطوط
