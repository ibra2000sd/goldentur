## ملخص التعديلات الرئيسية

تم تحويل الثيم الافتراضي "رائد" (Raed) إلى ثيم مخصص "الطور الذهبي" بهوية بصرية فاخرة تعكس متجر لحوم مستوردة عالية الجودة.

### 1. ملفات البنية الأساسية

#### `package.json`
- **التغيير:** تحديث اسم الثيم والوصف
- **من:** `"name": "theme-raed"`
- **إلى:** `"name": "theme-altour-aldahabi"`
- **السبب:** تحديد هوية الثيم الجديد

#### `twilight.json`
- **التغيير:** تحديث معلومات الثيم
- **التفاصيل:**
  - اسم الثيم: "الطور الذهبي" (Al-Tour Al-Dhahabi - Premium Meats)
  - رابط المستودع الجديد
  - الحفاظ على جميع الميزات والإعدادات الأصلية

## 2. ملفات الأنماط والمتغيرات

### `src/assets/styles/01-settings/global.scss`
**التغيير الرئيسي:** نظام ألوان فاخر جديد

**الألوان الجديدة:**
```scss
--black:        #080807;      // أسود عميق
--charcoal:     #111110;      // فحمي
--ember:        #1A0A05;      // لون الجمر
--red:          #9B1C1C;      // أحمر داكن (اللون الأساسي)
--red-hot:      #C42B2B;      // أحمر ساخن (عند التمرير)
--gold:         #C9973A;      // ذهبي (اللون المميز)
--gold-lt:      #E5BE7A;      // ذهبي فاتح
--cream:        #F2EAD8;      // كريمي (النص الأساسي)
--bone:         #D4C9B0;      // عظمي (نص ثانوي)
--smoke:        #7A7468;      // دخاني (نص خافت)
```

**المتغيرات الإضافية:**
- متغيرات الظلال (shadows)
- متغيرات الانتقالات (transitions)
- متغيرات الحدود (borders)

### `src/assets/styles/01-settings/fonts.scss` (جديد)
**الإضافة:** استيراد الخطوط الفاخرة

**الخطوط المستخدمة:**
- `Noto Naskh Arabic` - النص الأساسي (عربي)
- `Amiri` - العناوين والنصوص المعروضة (عربي)
- `Cormorant Garamond` - النصوص اللاتينية والعلامات

## 6. معايير التوافق

### RTL (اليمين إلى اليسار)
✅ تم الحفاظ على دعم RTL الكامل
- جميع العناصر تتجاوب مع اتجاه النص
- الهوامش والحشوات معكوسة تلقائياً عند الحاجة

### Responsive Design
✅ تصميم متجاوب كامل
- **Desktop:** شاشات كبيرة (1280px+)
- **Tablet:** شاشات متوسطة (768px - 1279px)
- **Mobile:** شاشات صغيرة (< 768px)

### التوافق مع Salla
✅ الحفاظ على التوافق الكامل
- جميع مكونات Salla محفوظة
- لا تغيير في البنية الديناميكية
- جميع الإعدادات الأصلية محفوظة

## 9. خطوات التثبيت والاستخدام

### المتطلبات
- Node.js 14+
- pnpm (Package Manager)

### التثبيت
```bash
cd theme-altour-aldahabi
pnpm install
```

### البناء
```bash
# وضع التطوير
pnpm run development

# وضع الإنتاج
pnpm run production

# المراقبة المستمرة
pnpm run watch
```

### التثبيت في Salla
1. رفع الثيم إلى لوحة Salla
2. تفعيل الثيم من الإعدادات
3. تخصيص الألوان والخطوط من لوحة التحكم
